Deno: A Modern Runtime for JavaScript and TypeScript

Deno is a secure runtime for JavaScript and TypeScript, created by the original developer of Node.js. It is built on the V8 JavaScript engine and Rust programming language.

Key Features of Deno

  • Integrated support for TypeScript
  • Security-first approach (e.g., file, network, and environment access must be explicitly enabled)
  • No need for a package manager like npmâ€”Deno uses ES module imports directly from URLs
  • Supports WebAssembly and the Fetch API natively
  • Can import npm packages starting from Deno v2

Why We Chose Deno

Deno was selected for this project due to its modern approach to JavaScript/TypeScript development, increased security measures, and seamless integration with ES modules. This makes it easier to manage dependencies and work without needing a node_modules folder or package manager.
