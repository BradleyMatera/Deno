Frontend Technologies: HTML, CSS, and JavaScript

The frontend of this project is built using modern web development technologies such as HTML, CSS, and JavaScript to ensure a fast, responsive, and accessible interface.

HTML5

HTML5 is the latest version of Hypertext Markup Language, allowing us to structure the web content effectively. It includes semantic elements that improve accessibility and SEO, ensuring our content is clear and well-structured.

CSS3

CSS3 is used to style the pages, applying the provided color scheme and ensuring the design is responsive across devices. The following custom colors are used:

  • Amaranth Purple: #a4243b
  • Ecru: #d8c99b
  • Butterscotch: #d8973c
  • Alloy Orange: #bd632f
  • Charcoal: #273e47

JavaScript (ES6+)

JavaScript adds interactivity to the page. It is primarily used for handling DOM manipulation, form validation, and making AJAX requests to the backend API (powered by Express).
