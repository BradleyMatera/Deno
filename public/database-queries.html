MongoDB & Dex ORM Integration

In this project, MongoDB is used as the database, and Dex ORM is responsible for managing the queries. This ensures efficient database operations with clean and maintainable code.

Connecting to MongoDB


// Import the Dex ORM and MongoDB libraries
import { Dex } from "https://deno.land/x/dex/mod.ts";
import { MongoClient } from "https://deno.land/x/mongo/mod.ts";

// Set up the MongoDB connection
const client = new MongoClient();
client.connect("mongodb://localhost:27017");

// Define the Dex ORM query builder for MongoDB
const dex = Dex({ client: "mongodb" });

Basic CRUD Operations with Dex ORM

Dex ORM allows us to write simple and maintainable SQL-like queries for MongoDB. Here are a few examples:


// Insert a new user
await dex.table("users").insert({
  name: "John Doe",
  email: "john@example.com",
});

// Fetch all users
const users = await dex.table("users").select();

// Update a user
await dex.table("users").where("id", 1).update({ name: "Jane Doe" });

// Delete a user
await dex.table("users").where("id", 1).del();
